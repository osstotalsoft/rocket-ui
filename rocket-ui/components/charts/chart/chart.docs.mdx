## Basic Chart component

Basic example that renders a chart

### Component usage

```js
<Chart
  title='Line chart'
  subheader={'Chart infos'}
  type='line'
  icon={BarChartIcon}
  iconColor='info'
  data={lineChartData}
  options={chartOptions}
/>
```
