## Basic CollapseCard component

Basic example that renders an collapse-card

### Component usage

```js
<CollapseCard
  title='Delivery information'
  actions={
    <IconButton key='1' color='info' variant='text' size='small'>
      <QuestionMark />
    </IconButton>
  }
  content={
    <Grid container rowSpacing={3} justifyContent='center'>
      <Grid item xs={8}>
        <Typography variant='body1'>Please, fill in with your personal information.</Typography>
      </Grid>
    </Grid>
  }
/>
```
