## Basic Pagination component

Basic example that renders a pagination

### Component usage

```js
<Pagination
  component='div'
  count={100}
  page={0}
  onPageChange={() => {}}
  pageSize={25}
  onRowsPerPageChange={() => {}}
  rowsPerPageOptions={[25, 50, 75, 100]}
/>
```
